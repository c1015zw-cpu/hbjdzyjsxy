# 河北机电职业技术学院迎新导航

面向河北机电职业技术学院 2026 级新生的手机优先静态网站。页面为非官方经验整理，年度报到安排以学校正式通知为准。

## 页面内容

- 官方入口：学校官网、招生信息网、迎新系统、智慧校园和招生咨询电话
- 新生行动：报到四步流程、五组阶段任务和第一周安排
- 随手工具：可筛选服务目录、浏览器本地清单与完成进度
- 安全答疑：开学季防骗提醒、常见问题和微信联系二维码
- 手机体验：底部导航、七色主题、可拖动挂坠、减少动态偏好和响应式布局

## 本地预览

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

浏览器打开 `http://127.0.0.1:4173/`。

## 信息边界

地址、公开电话和官方入口于 2026-07-21 从学校官网公开页面核验。报到日期、缴费、宿舍、军训和接站等年度安排不得由本页面替代学校正式通知。
