# 河北机电职业技术学院迎新导航

静态单页迎新网站，面向河北机电职业技术学院 2026 级新生。页面整理报到流程、到校前任务、宿舍准备、官方入口、入学清单和防坑提醒。

## 页面模块

- 第一屏：学校迎新标题、账号视觉、学校地址和招生咨询电话
- 报到路线：到校当天四步流程
- 任务卡：到校前、报到日、宿舍、开课前、避坑五类内容
- 入学账本：本地保存的勾选清单
- 官方入口：官网、招生网、迎新系统、智慧校园

## 说明

本页面为非官方新生攻略页。报到时间、缴费、宿舍、军训、接站安排等高风险信息，必须以河北机电职业技术学院当年正式通知为准。
